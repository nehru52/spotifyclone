# Settings Screens - User Guide

## 📱 How to Access

From the **Profile** tab (bottom right), tap any of these options:

1. **Downloads** - Manage offline tours
2. **GPS Auto-Play** - Configure location tracking
3. **Notifications** - Notification preferences ✨ NEW
4. **Language** - App language selection ✨ NEW
5. **Support** - Get help and contact us ✨ NEW
6. **About** - Learn about the app ✨ NEW

---

## 🔔 Notifications Screen

### What You'll See
- **Notification Types** section with toggles for:
  - 📱 Push Notifications (master toggle)
  - 📍 Location Alerts (GPS notifications)
  - 📥 Download Complete
  - ⏰ Tour Reminders
  - 🏷️ Promotions & Offers

- **Sound & Vibration** section:
  - 🔊 Sound toggle
  - 📳 Vibration toggle

- **Test Notification** button to preview notifications

### How to Use
1. Tap any toggle to enable/disable that notification type
2. Settings save automatically
3. Tap "Send Test Notification" to see how notifications look
4. Tap back arrow to return to Profile

### Tips
- Disable "Promotions" if you only want essential notifications
- Keep "Location Alerts" on for GPS Auto-Play to work best
- Test notifications to make sure they're working

## 🔧 Troubleshooting

### Notifications Not Working
1. Check if push notifications are enabled in device settings
2. Make sure "Push Notifications" toggle is ON in app
3. Try sending a test notification
4. Restart the app

### Language Not Changing
1. Some changes require app restart
2. Remember: App language ≠ Audio language
3. Check if language is properly selected (checkmark visible)

### Contact Links Not Opening
1. Make sure you have the app installed (WhatsApp, email)
2. Check internet connection for website
3. Grant necessary permissions if prompted

### Can't Find a Setting
1. All settings are in Profile → Settings section
2. Scroll down to see all options
3. Downloads and GPS are at the top
