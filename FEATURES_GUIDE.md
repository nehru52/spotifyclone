# Rome Audio Tour App - Features Guide

## 🎧 Offline Downloads

### How to Download Tours
1. Go to **Home** screen
2. Find any audio tour card
3. Tap the **download button** (top-right corner of card)
4. Wait for download to complete (shows progress %)
5. Green checkmark appears when done

### Download States
- 📥 **Download Icon**: Not downloaded yet
- ⏳ **Spinner + %**: Downloading in progress
- ✅ **Green Checkmark**: Downloaded and available offline

### Managing Downloads
1. Go to **Profile** → **Downloads**
2. View all downloaded tours
3. See storage usage
4. Delete individual tours or all at once
5. Tap any tour to play it

### Automatic Offline Playback
- App automatically uses offline files when available
- Falls back to streaming if not downloaded
- No manual switching needed!

## 📍 GPS Auto-Play (Geofencing)

### How to Enable GPS Tracking
**Option 1: From Map Screen**
1. Go to **Map** tab
2. See GPS toggle card at top
3. Tap the switch to enable

**Option 2: From Settings**
1. Go to **Profile** → **GPS Auto-Play**
2. Enable "GPS Tracking"
3. Enable "Auto-Play" (optional)
4. Enable "Notifications" (optional)
5. Adjust detection radius (25-200m)

### How It Works
1. **GPS Tracking**: Monitors your location in background
2. **Geofencing**: Detects when you enter a tour location zone
3. **Auto-Play**: Automatically starts audio when you're near (if enabled)
4. **Notifications**: Alerts you when near a location (if enabled)
5. **Cooldown**: 5-minute cooldown to prevent repeated triggers

### 11 Tracked Locations
1. Colosseum
2. Roman Forum
3. Heart of Rome
4. Jewish Ghetto
5. Ostia Antica
6. Pantheon
7. Sistine Chapel
8. St. Peter's Basilica
9. Trastevere
10. Vatican Museums
11. Vatican Pinacoteca

### GPS Settings
- **GPS Tracking**: Enable/disable location monitoring
- **Auto-Play**: Automatically play audio when near location
- **Notifications**: Get alerts when near locations
- **Detection Radius**: 25m, 50m, 100m, 150m, or 200m

### GPS Status Indicators
- 🟢 **Green Icon**: GPS tracking active
- ⚪ **Gray Icon**: GPS tracking disabled
- 📍 **"Active - Tracking your location"**: GPS is working
- 🔴 **"Tap to enable"**: GPS is off

## 💡 Tips & Tricks

### For Best Experience
1. **Download tours before visiting** - Avoid using mobile data
2. **Enable GPS before arriving** - Get automatic playback
3. **Adjust detection radius** - Larger radius = earlier triggers
4. **Keep app in background** - GPS works even when app is closed
5. **Grant location permissions** - Required for GPS features

### Battery Saving
- Disable GPS when not touring
- Download tours on WiFi
- Use smaller detection radius
- Disable notifications if not needed

### Troubleshooting
- **App stuck on splash**: Wait 2-3 seconds for GPS to initialize
- **GPS not working**: Check location permissions in device settings
- **Downloads failing**: Check internet connection and storage space
- **Audio not playing**: Check if tour is downloaded or internet is available
